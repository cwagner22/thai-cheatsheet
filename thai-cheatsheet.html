How to Remember the Classes

21 sounds split into 3 groups of 7. Ask yourself one question about the sound:

(each sound has both — check the tables above)

Extra tips for the high/low pairs:

• Most sounds have one high-class letter and two+ low-class letters — when in doubt, low is the safer guess.

• The high-class letter usually comes first in the alphabet (e.g. ข before ค, ฉ before ช, ถ before ท).

• The consonant names are designed so the tone reveals the class — e.g. ส เสือ (rising tone) must be high class because a low-class consonant can't produce a rising tone without a tone mark.

Other Notes

Only 8 final sounds exist in Thai: /k/, /t/, /p/ (stops) and /ŋ/, /n/, /m/, /j/, /w/ (sonorants). Many different initial consonants collapse to the same final.

ห นำ (leading ห): Low-class sonorants (น ม ง ย ร ล ว) have no high-class letter. Putting a silent ห in front bumps them to high-class tone rules. That's all it does — ห is silent, it just changes the tone.

Tips

When in doubt, guess low class — most sounds have more low-class letters than high-class.

The consonant name reveals the class — e.g. สอ เสือ (rising tone) must be high class, because low-class consonants can't produce a rising tone without a tone mark.

Classical class mnemonics (ไตรยางศ์):

Mid Class (9): ไก่ จิก เด็ก ตาย บน ปาก โอ่ง — "A chicken pecks a dead child on top of a jar"
→ ก  จ  ฎ ฏ  ด ต  บ  ป  อ
High Class (11): ผี ฝาก ถุง ข้าว สาร ให้ ฉัน — "A ghost entrusts a bag of rice to me"
→ ผ  ฝ  ถ ฐ  ข ฃ  ส ศ ษ  ห  ฉ
Low Class (24): everything not in the Mid or High mnemonic — memorized by elimination.

Vowel Tips

Short vs long matters for tones: dead syllable + low class: short → high tone, long → falling tone.

-ั (mai han akat) replaces -ะ when a final consonant closes the syllable: กะ → กัน.

ใ- only in 20 words. Everything else uses ไ-. If in doubt, use ไ-.

Tone Calculation

Start with the initial consonant's class, then follow these steps:

Step 1 — Is there a tone mark?
→ Yes: check the tone mark table below. The mark + class = your tone.

Step 2 — No tone mark. Is the syllable live?
→ Mid or Low class → mid tone
→ High class → rising tone

Step 3 — No tone mark, dead syllable.
→ Mid or High class → low tone (short or long vowel, doesn't matter)
→ Low class + short vowel → high tone
→ Low class + long vowel → falling tone

Live vs Dead:

Live = you can hold it — ends in a sonorant (ง น ณ ม ญ ร ล ฬ ย ว) or an open long vowel.
Dead = ends abruptly — ends in a stop (/k/, /t/, /p/) or a short vowel with no final.

The other two tone marks (Mid class only):

• ก๊ ไม้ตรี → High tone.

• ก๋ ไม้จัตวา → Rising tone.

Tips:

• ไม้เอก = Dead-long behavior for every class. Memorize one, get both.

• Only Mid class uses all 4 tone marks and can produce all 5 tones.

• With a tone mark, the result is never Mid tone.

Leading Consonants — อักษรนำ
Silent ห or อ shifts tone class for sonorants

Why? Sonorants (ง น ม ย ร ล ว) are always low class, so they can only produce mid, high, and falling tones on their own. Prepending a silent ห (high class) unlocks rising and low tones.

Non-Conforming Clusters
Two consonants that don't form a true cluster — an unwritten /a/ is inserted

When two consonants can't merge into a true cluster, a short /a/ sub-syllable is inserted between them. This /a/ is not written. No tone mark can be used on these.

ขนม → /kʰa-nom/ (snack)  •  สนุก → /sa-nuk/ (fun)  •  แขนง → /kʰa-nɛːŋ/ (branch)

Some clusters produce /aː/ (long) instead: มรกต → /mɔː-ra-got/ (emerald)  •  ทราบ → handled as false cluster above

Cluster Rules

Tone rule for clusters:

If 2nd consonant is a sonorant (ง น ม ย ร ล ว):
→ Tone is determined by the 1st consonant's class
ปลูก /bpluːk/ — low tone from mid-class ป  •  สนุก /sa-nuk/ — low tone from high-class ส

If 2nd consonant is NOT a sonorant:
→ Tone is determined by the 2nd consonant's class
แสดง /sa-dɛːŋ/ — mid tone from mid-class ด  •  เฉพาะ /tɕʰa-pʰɔ/ — low tone from low-class พ

Orthography:

Preposed vowels go before the entire cluster: โปรด not ปโรด
Tone marks & superscript vowels sit above the 2nd consonant: กล้อง (mark on ล not ก)
Subscript vowels go under the 2nd consonant: พริ้ม (อิ under ร not พ)
